# 📚 Web Development Teaching Slides

Welcome to my presentation repository! This is the home for the slide decks I use to teach web development. You can view the live files and structure directly in the [seanwhs/Slides GitHub repository](https://github.com/seanwhs/Slides/tree/main).

My core teaching philosophy is simple: **no flashing animations, no distracting templates—just clean, plain slides focused entirely on clear code and core concepts.** I intentionally use a minimalist approach so my students can focus fully on learning how to program without unnecessary visual noise.

## 🎯 About These Slides

I design these decks to serve as practical, high-utility teaching aids for classrooms and code walkthroughs:
* **High Contrast:** Clean layouts that are easy to read from the back of a lecture hall or over a low-resolution screen share.
* **Code-First:** Packed with syntax examples, architectural logic, and explicit step-by-step breakouts.
* **Distraction-Free:** Plain, minimalist styling so the focus remains entirely on the educational content.

## 🛠️ How I Created Them

I build these presentations using **LibreOffice Impress**. I maintain the raw OpenDocument Presentation (`.odp`) files here alongside standard PDF exports. This approach makes it incredibly simple to either present them as-is or download and modify them for your own instructional style.

## 💻 How to Use These Resources

### For Students
If you are using these decks to study or review a lecture, I highly recommend checking out the **PDF** versions. They render perfectly across all devices and are great for self-paced learning.

### For Educators
If you want to adopt or adapt my slides for your own classes:
1. **Clone or download** the repository using the [main branch](https://github.com/seanwhs/Slides/tree/main).
2. Open the OpenDocument Presentation (`.odp`) files in **LibreOffice**.
3. Feel free to tweak the examples, reformat the layouts, or integrate your own coding challenges.
